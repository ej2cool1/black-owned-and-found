# Black-Owned & Found

Black-Owned & Found is a full-stack web application designed to help users discover and support Black-owned businesses. The platform allows users to search for businesses, submit new listings, and contribute to a growing directory of businesses.

## Features

- Search for Black-owned businesses
- Submit new businesses to the platform
- User account creation and login
- Backend API for managing business data
- Database storage for business listings
